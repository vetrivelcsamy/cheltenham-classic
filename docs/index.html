Using Cheltenham Classic is as easy as downloading & installing the font files.
If you're making a web thing, you can use the following HTML and CSS:

<link rel="preconnect" href="https://github.io/">
<link rel="stylesheet" href="https://vetrivelcsamy.github.io/cheltenham-classic/cheltenhamclassic.css">

Use the following CSS rules to specify the cheltenham classic font family:

font-family: 'cheltenhamclassic', serif;


Free & Open Source

Cheltenham Classic Typeface is a free and open source font family.

You are free to use this font in almost any way imaginable.

Refer to the SIL Open Font License 1.1 for exact details on what the conditions and restrictions are.

The story behind Cheltenham Classic

Cheltenham Classic was born out of a desire to create a high-quality, freely available typeface inspired by the timeless elegance of the original Cheltenham font. Our goal was to make a versatile and beautiful typeface accessible to designers, writers, and creators worldwide without the burden of licensing fees or restrictions.

Current status & usability

Cheltenham Classic is currently in beta, with over 150+ glyphs available. At present, it supports only the English language. We are continuously working on expanding the character set and improving the overall design.

Frequently Asked Questions

  • Is Cheltenham Classic completely free to use?

    Yes, Cheltenham Classic is open source and free for personal and commercial use.

  • Can I modify the typeface?

    Absolutely! As an open source project, you're welcome to modify and adapt the typeface to suit your needs.

  • Will other languages be supported in the future?

    We plan to expand language support in future releases, but we don't have a specific timeline yet.

  • How can I contribute to the project?

    We welcome contributions! Please check our GitHub repository for guidelines on how to get involved.

  • Is Cheltenham Classic suitable for print and web use?

    Yes, it's designed to work well in both print and digital environments.

  • How often is the typeface updated?

    We aim to release updates regularly, but the frequency may vary depending on the scope of improvements and available resources.
